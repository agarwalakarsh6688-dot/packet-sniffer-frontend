# 🔎 Live TCP Packet Sniffer

A **cybersecurity-inspired web dashboard** that simulates real-time TCP packet monitoring through a live backend API.

This project demonstrates **network traffic visualization, API integration, and a cyber-themed monitoring interface** built with modern web technologies.

## 🧠 Project Overview

The application simulates a **network monitoring system** that continuously fetches TCP packet data from a backend API and displays it in a **real-time cyber dashboard**.

It features an animated **network topology background**, live packet updates, and a terminal-style UI inspired by cybersecurity monitoring tools.

---

## ✨ Features

* 🌐 Real-time packet monitoring
* 🧠 Live API integration
* 📊 Dynamic packet table
* 🎛 Start / Stop monitoring controls
* 🧹 Clear table & reset interface

## ⚙️ How It Works

1. User clicks **Start Monitoring**
2. Frontend begins polling backend API
3. Backend generates simulated packet data
4. Packets are streamed to the dashboard
5. Table updates every second
